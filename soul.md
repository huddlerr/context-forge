# Soul — North Star

## The One Thing
Context Forge gives AI coding agents the context they need to work effectively. Two modes: (1) Project Scaffold — 10 questions → 10 production-ready config files (soul.md, claude.md, package.json, types.ts, etc.); (2) Second Brain — 6 onboarding sections → full personal executive assistant folder structure for Claude Code (CLAUDE.md, context/, .claude/rules/, projects/, decisions/).

## Who
Solo developers and small teams (ages 20-45) who use AI coding assistants like Claude, Cursor, or Copilot. They need structured context files so their AI agents understand the project from day one.

## Voice
- Teammate, not manual
- Warm but efficient
- Errors help, never blame

## Test: "Does this serve THE ONE THING?"
> No → cut. Maybe → defer. Yes → build beautifully.
